## 这是什么

Zhrradiant SrvMap 是一个面向 L4D2 玩家的 Windows 桌面工具。你不用打开浏览器，就能快速浏览服务器状态、找到想玩的服、一键连接进游戏。

核心思路很简单：**找服 → 看状态 → 一键进游戏**。

---

## 功能一览

### 服务器浏览

- **服务器组** — 由站点用户创建和维护的服务器列表，支持组收藏，快速定位你关注的服务器组
- **公共服务器** — 全部公开服务器，支持搜索、筛选（在线/空闲、官图/三方图、游戏模式）、按玩家数排序、分页
- **组详情** — 展示组的描述、头像、自定义按钮和图片

### 服务器状态查询

默认通过云端获取服务器实时信息，也可切换为本地模式（A2S 协议直连）获取更低延迟的数据：

- 在线状态、当前地图、玩家数 / 上限
- 在线玩家列表（名称、分数、在线时长）
- 网络延迟、游戏模式、VAC 状态、服务器版本

支持一键切换"云端"和"本地"两种查询模式。

### 热门统计

- 近 3 天热门三方地图排行榜（Top 20）
- 排名变化趋势标记（上升 / 新上榜）
- 热门服务器排行
- 点击排行榜上的地图名可直达地图详情

### 地图模块

一个完整的地图社区系统：

- **地图库** — 浏览三方地图，搜索、按标签/标记筛选
- **标签系统** — 彩色标签标注地图特性（写实、多特、变态等）
- **评分** — 5 星制评分，显示平均分和评分人数
- **评论** — 地图评论，支持回复
- **收藏夹** — 创建多个收藏夹分类管理地图，支持批量操作
- **分享码** — 将收藏夹生成为分享码发给朋友，对方可一键导入
- **在线状态** — 查看哪些服务器当前正在运行某张地图
- **下载链接** — 直达 Steam Workshop 下载页
- **个人标记** — 标记"玩过""喜欢""不喜欢"

### 服务器收藏

- 创建多个收藏夹，分类管理常玩的服务器
- 从公共服务器、组服务器、历史记录中一键添加
- 批量操作（多选添加/移除）
- 服务器收藏分享码（和朋友分享你的服务器列表）

### 连接历史

- 自动记录通过客户端连接过的每一台服务器
- 逐条移除或一键清空
- 基于历史记录重新浏览和连接，不用再翻找

### 下载管理

- 地图文件下载队列，支持暂停和取消
- 实时显示进度和下载速度
- 自定义下载目录，一键打开目录

### 自动更新

- 启动时静默检查是否有新版本
- 弹窗提示更新内容和版本号

### 一键进游戏

点击任意服务器卡片查看详情，再点"连接服务器"，自动唤起 Steam 并进入游戏。不需要手动复制 IP。

### 其他亮点

- 云端数据，与竹烨柠檬站点打通
- 服务器详情分享链接一键复制
- 无边框窗口 + 圆角 + 自定义标题栏
- 侧边栏导航，模块一目了然

## 安装与使用

### 下载

前往 [Releases](https://github.com/Zhrradiant/zhrradiant-srvmap/releases) 页面，下载最新版本的 `Zhrradiant SrvMap.exe`。

### 使用

1. 双击运行
2. 使用竹烨柠檬的站点账号进行登录（没有账号可点击"注册"跳转）
3. 浏览服务器 → 点击查看详情 → 一键连接进游戏

### 系统要求

- Windows 10 及以上
- Steam 客户端（用于连接服务器）
